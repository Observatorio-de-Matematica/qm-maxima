1.1 Introduction to package qm ¶

Package version: 0.5

The qm package provides functions and standard definitions to solve quantum mechanics problems in a finite dimensional Hilbert space. One can calculate the outcome of Stern-Gerlach experiments using the built-in definition of the Sx, Sy, and Sz for arbitrary spin, e.g. s={1/2, 1, 3/2, …}. One can create ket vectors with arbitrary but finite dimension and perform standard computations. Angular momentum representation of kets is available as well.

With this package it is also possible to create tensor product states for multiparticle systems and to perform calculations on those systems.

The package is loaded with: load(qm);
